Version Control Systems

Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. A Version Control System (VCS) is very useful in the world of a designer. VCS gives you the ability to return files back to their previous state, return the entire project back to a previous state, compare the changes you have made throughout the project and see who last modified something that might be causing an issue in the design. In other words, VCS is a way you can revert back in the case of losing files or messing them up.

Github

GitHub is a web based Git or version control system and Internet hosting service. Along with providing the standard version control and source code management functionality, it has its own capabilities as well. Such as feature requests, task management, and bug tracking. GitHub is mostly used to host open-source software projects. To access projects on GitHub, the user can use a standard Git command-line and all the regular commands that go with it. GitHub also provides a great way to network. The site has social networking functions such as feeds and followers. This is a way developers can show others what projects they have worked on.

Git Commands

The Git command line comes standard with a lot of different commands. When you first use GitHub, one thing you will want to do is tell Git who you are. To do this you would use the ‘git config’ command, followed by your username and email. To create a repository, you would use the ‘git init’ command. The way you add files to these repositories requires three main commands. The first one is ‘git add *’, which adds whatever directory you are in to git. The next command ‘git commit –(message)’ allows you to commit any files you added to git. The last command ‘git push’ or ‘git push origin master’ send the changes or new files to the repository you choose or the master branch of your remote repository.

Repository

A repository or an Information repository is a way of having secondary data storage that is capable of creating multiple networked data storage technologies that are running on diverse operating system. The primary data doesn’t need to have such a protective storage security due to this even though it is still suggested. Information repositories were created to solve the problems that were coming from data proliferation and get rid of the need for separate data storage solutions. Repositories all have centralized management. They are also self-contained storage resources that add, maintain, recycle, and can terminate any of the inputted data.

Advantages of Github

GitHub is a very beneficial way to store data. Even though these are all opinions, one of the main advantages is that it can help you become a better developer. You can go through different peoples’ code with also the option of creating new things while showing it to others around you. Another benefit is that is it open source. Open source allows the original source code to be made freely available and may be reused and changed by other people. Another great advantage is that you can show off your projects and code to your work or possible recruiters of bigger companies. It provides a storage option as well as a social aspect to show off what you can accomplish.
